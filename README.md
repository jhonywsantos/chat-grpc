# chat-grpc
This project was developed as part of the Distributed Systems course. It consists of implementing a chat application using gRPC, in which messages can be sent and received simultaneously. In other words, the communication is asynchronous, allowing users to send messages without needing to wait for incoming ones, ensuring a more dynamic and real-time interaction.
